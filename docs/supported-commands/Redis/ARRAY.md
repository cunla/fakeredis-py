
## Unsupported array commands
> To implement support for a command, see [here](/guides/implement-command/)

#### [ARCOUNT](https://redis.io/commands/arcount/) <small>(not implemented)</small>

Returns the number of non-empty elements in an array.

#### [ARDEL](https://redis.io/commands/ardel/) <small>(not implemented)</small>

Deletes elements at the specified indices in an array.

#### [ARDELRANGE](https://redis.io/commands/ardelrange/) <small>(not implemented)</small>

Deletes elements in one or more ranges.

#### [ARGET](https://redis.io/commands/arget/) <small>(not implemented)</small>

Gets the value at an index in an array.

#### [ARGETRANGE](https://redis.io/commands/argetrange/) <small>(not implemented)</small>

Gets values in a range of indices.

#### [ARGREP](https://redis.io/commands/argrep/) <small>(not implemented)</small>

Searches array elements in a range using textual predicates.

#### [ARINFO](https://redis.io/commands/arinfo/) <small>(not implemented)</small>

Returns metadata about an array.

#### [ARINSERT](https://redis.io/commands/arinsert/) <small>(not implemented)</small>

Inserts one or more values at consecutive indices.

#### [ARLASTITEMS](https://redis.io/commands/arlastitems/) <small>(not implemented)</small>

Returns the most recently inserted elements.

#### [ARLEN](https://redis.io/commands/arlen/) <small>(not implemented)</small>

Returns the length of an array (max index + 1).

#### [ARMGET](https://redis.io/commands/armget/) <small>(not implemented)</small>

Gets values at multiple indices in an array.

#### [ARMSET](https://redis.io/commands/armset/) <small>(not implemented)</small>

Sets multiple index-value pairs in an array.

#### [ARNEXT](https://redis.io/commands/arnext/) <small>(not implemented)</small>

Returns the next index ARINSERT would use.

#### [AROP](https://redis.io/commands/arop/) <small>(not implemented)</small>

Performs aggregate operations on array elements in a range.

#### [ARRING](https://redis.io/commands/arring/) <small>(not implemented)</small>

Inserts values into a ring buffer of specified size, wrapping and truncating as needed.

#### [ARSCAN](https://redis.io/commands/arscan/) <small>(not implemented)</small>

Iterates existing elements in a range, returning index-value pairs.

#### [ARSEEK](https://redis.io/commands/arseek/) <small>(not implemented)</small>

Sets the ARINSERT / ARRING cursor to a specific index.

#### [ARSET](https://redis.io/commands/arset/) <small>(not implemented)</small>

Sets one or more contiguous values starting at an index in an array.
